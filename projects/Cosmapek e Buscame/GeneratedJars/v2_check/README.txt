 QUICK START
---------------

1) Delete the Databases.

2) Start the server by executing “server.sh” (or “server.bat” if you’re using
   Windows) under the “bin” directory.


3) When the database server is ready, use different consoles to execute the servers.

	A example to start a sever:
	-create a new console
	-cd to the directory
	-execute java -jar buscameApi_A_1723.jar in the console.

4) Wait to the serves are ready.
